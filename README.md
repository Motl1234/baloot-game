# 🃏 ديوانية بلوت

لعبة بلوت سعودية كاملة تعمل في المتصفح — ملف HTML واحد، بدون سيرفر وبدون أي تثبيت.

## ✨ المميزات

- **لعب فردي**: أنت وشريك ذكاء اصطناعي ضد خصمين، بذاكرة للورق الملعوب
- **10 مستويات صعوبة**: ترتفع تلقائيًا كل ما فزت بصكّة وتنزل إذا خسرت، مع إمكانية الاختيار اليدوي
- **لعب جماعي عن بعد بغرف** 🎪: المضيف ينشئ غرفة برمز من 4 حروف عبر Cloudflare Worker، وينضم حتى 3 أصدقاء بالرمز فقط (أول منضم = شريكك)
- **طريقة يدوية بديلة بدون سيرفر**: اتصال مباشر بين جهازين (WebRTC) بتبادل رمزين — تعمل حتى بدون خادم الغرف
- **قوانين بلوت الصحيحة**: شراء على جولتين (حكم/صن)، اتباع النوع، الترفيع والقصّ الإجباري، آخر أكلة +10
- **الحساب الكامل**: 16 بنط حكم / 26 صن (مضاعفة)، خسارة الشاري، كبوت (25/44)، الصكّة حتى 152
- **المشاريع**: سرا، خمسين، مية، أربعمية، بلوت — مع خيار الإعلان أو الإخفاء ومقارنة المشاريع بين الفريقين
- **مساعدات للمبتدئ**: شريط يشرح القانون المطبّق الآن، الأوراق الممنوعة باهتة مع سبب المنع، أزرار «وش تنصحني؟» و«اقترح لي ورقة»
- **حسبة واضحة**: شاشة نهاية الجولة تشرح الحساب خطوة بخطوة مع مضمار سباق نحو 152
- **إيموجي أثناء اللعب** و**دردشة بعد كل جولة** (في اللعب عن بعد)

## 🎮 طريقة التشغيل

افتح `index.html` في أي متصفح حديث، أو العب مباشرة عبر GitHub Pages.

### اللعب الجماعي عن بعد (غرف)
1. المضيف: «لعب عن بعد» ← «🎪 أنشئ غرفة» ← شارك رمز الغرفة (4 حروف)
2. كل صديق: يكتب الرمز ← «انضم للغرفة»
3. المضيف: «🚀 ابدأ اللعب» — من ينقطع يكمّل الذكاء مكانه، ويقدر يرجع بنفس الرمز

### الطريقة اليدوية (بدون خادم)
تحت زر «الطريقة اليدوية»: تبادل رمزَي دعوة/ردّ عبر واتساب ← اتصال مباشر (WebRTC).

## 🛠 التقنية

- اللعبة: HTML + CSS + JavaScript خام في ملف واحد (`index.html`) — لا مكتبات
- خادم الغرف: Cloudflare Worker + Durable Objects (`worker/`) — موزّع رسائل WebSocket فقط، منطق اللعبة كله عند المضيف
- نشر الخادم: `cd worker && npx wrangler deploy` (ثم حدّث `RELAY_URL` في `index.html`)
- البديل اليدوي: WebRTC DataChannel بتبادل يدوي لرموز الاتصال
